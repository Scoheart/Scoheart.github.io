如何建立自己的blog

1.下载安装Node.js

2. 建立github账户 并且 创立repository

3.下载安装Git

  • ssh-keygen -t rsa -C “Scoheart1487790559mail.com”

4.Github ssh配置

5.Hexo本地搭建

  • npm install hexo-cli -g
  • hexo init blog(hexo init Scoheart)
  • cd blog(cd Scoheart)
  • npm install
  • hexo server(hexo s -p 5555)
