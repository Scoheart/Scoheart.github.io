博客后续跟进

发表于 03-08-2021 更新于 03-09-2021 分类于 blog
添加标签、分类等。

添加分类

  • 根目录下 git bash
  • hexo new page categories
  • 给文章加上categories: xxx

添加标签

  • 根目录下 git bash
  • hexo new page tags
  • tags: xxx
  • 给文章加上categories: xxx

更换主题

  • 浏览主题
  • 点击喜欢的主题的标题进入github下载界面
  • 下载主题源代码目录
  • 移至博客themes文件夹下并改名
  • 将站点配置文件中的theme改为将要使用的主题

添加社交网站链接

  • 进入主题配置文件
  • 在social标签下添加各种社交网站链接

添加社交网站链接图标

  • 进入主题配置文件
  • 在网站链接后面加 || fab fa-xxxxx

主页文章显示摘要显示

  • 文章Fromt-matter部分加上

description: xxxxx

设置站点地图

  • 根据下面代码下载npm插件

npm install hexo-generator-sitemap --save

  • 修改站点配置文件中的 Extensions 标签

  • 在主题配置文件中的menu标签添加sitemap项

添加相册

  • 在主题配置文件中的menu菜单中添加photos项
  • 在主题languages - zh-CN.yml文件中的menu标签添加photos: 相册

更改博客标题栏图片

  • 修改主题配置文件favicon标签下的图片路径
