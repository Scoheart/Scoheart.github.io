    How to start a blog.

    发表于 03-08-2021 更新于 03-09-2021 分类于 blog 阅读次数： Disqus：
    Start a blog with Hexo.

    Install Node.js.

    Download Node.js from the official website.

    Check Node.js using the following code

    • node -v
    • npm -v

    Install Git.

    Download Git from the official website.

    Check Git using the following code

    • git –version

    Install Hexo.

    Download Hexo using the following code.

    • npm install -g hexo–cli

    Check Hexo using the following code

    • hexo -v

    Initialize Hexo.

    • Create a floder（eg. name= “blog”）

    • git bash/ cmd —— blog

    hexo init “floder” (eg. name= “myblog”)

    cd “floder”

    npm install

    Web page preview

    • hexo generate (Generating the static web page)

    • hexo server (The preview page)

    • http://localhost:4000

    Deploy to github pages

    • create a github account
    • cerate a new repository

    repository name= “username.github.io”

    • generate SSH in git bash

    git config –global user.name “yourname”

    git config –global user.email “youremail”

    Check using the following code

    • git config user.name
    • git config user.email

    ssh-keygen -t rsa -C “youremail”

    • get the ssh public key

    C/user/yourname/.ssh/id_rsa.pub

    • paste to github

    Check using the following code

    • ssh -T git@github.com
    • modify site file

    _config.yml

    deploy:

    ​ type: git

    ​ repo: https://github.com/yourname/yourname.github.io.git

    ​ branch: main

    • install deploy-git

    npm install hexo-deployer-git –save

    • deploy

    hexo generate

    hexo deploy
