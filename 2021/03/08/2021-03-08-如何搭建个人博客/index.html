如何搭建个人博客

发表于 2021-03-08 分类于 blog

一、使用Hexo搭建博客

1.安装Node.js。

从官网下载Node.js

注：使用下面的代码检查Node.js的安装

node -v

npm -v

2.安装Git。

从官网下载Git

注：使用下面的代码检查Git的安装

git --version

3.安装Hexo。

使用下面的代码下载Hexo

npm install -g hexo--cli

注：使用下面的代码检查Hexo的安装

hexo -v

4. 初始化Hexo。

  • 创建一个新文件夹 于其中搭建博客（例：文件夹 “blog”）
  • 在此文件夹下以Git bash 打开（或者cmd 等等）
  • 运行下列命令搭建博客

hexo init "floder" (例：文件夹 “myblog”)

cd "floder"

npm install

5. 个人博客页面预览

hexo generate (生成静态博客页面)

hexo server (预览静态博客页面)

  • 以此本地地址进入静态博客页面

http://localhost:4000

6. 部署博客页面至Github

  • 创建一个Github账户

  • 新建一个仓库（例：”Username.github.io” 注：必须用此格式）

  • 生成SSH秘钥

git config --global user.name "yourname"

git config --global user.email "youremail"

注：使用下面代码检查账户邮箱输入

git config user.name

git config user.email

ssh-keygen -t rsa -C "youremail"

  • 从下面路径获取秘钥

C/user/yourname/.ssh/id_rsa.pub

  • 将秘钥粘贴至Github

注：使用下面的代码检查秘钥的上传

ssh -T git@github.com

  • 修改站点配置文件中的deploy标签 （名为”_config.yml”的文件）

deploy:

​ type: git

​ repo: https://github.com/yourname/yourname.github.io.git

​ branch: main

  • 安装deploy-git

npm install hexo-deployer-git --save

  • 部署博客至Github

hexo generate

hexo deploy

  • 以此网页地址进入博客页面

https://1487790559.github.io
